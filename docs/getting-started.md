# Getting Started

## Prerequisites

- Git installed (v2.30+)
- A GitHub account
- Basic command line knowledge

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/ezou18/test-CodingAgent.git
   cd test-CodingAgent
   ```

2. Explore the project structure:
   ```bash
   ls -la
   ```

## Project Layout

| Directory  | Purpose                  |
|------------|--------------------------|
| `docs/`    | Project documentation    |
| `answers/` | Q&A and exercise answers |
| `audit/`   | Audit logs and records   |

## Next Steps

- Read the documentation in `docs/`
- Check out open issues on GitHub
- Create a branch and make your first contribution
